## 👀 화면 구성 요소
- 스플래시 스크린
- 음악 재생 화면
    - 재생 중인 음악 정보(제목, 가수, 앨범 커버 이미지, 앨범명)
    - 현재 재생 중인 부분의 가사 하이라이팅
    - Seekbar
    - Play/Stop 버튼
- 전체 가사 보기 화면
    - 특정 가사로 이동할 수 있는 토글 버튼
    - 전체 가사 화면 닫기 버튼
    - Seekbar
    - Play/Stop 버튼
